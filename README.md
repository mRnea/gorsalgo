# Görsel Algoritma
Çizge ve ağaç gibi veri yapıları üzerindeki algoritmaları görselleştirme amaçlı yazılım.

## Kurulum
```bash
apt install libsdl2-dev
make
```

## Kullanım
+ -load graph.txt
İçinde boyut ve komşuluk matrisi olan dosyayı aç.

+ -builder
Çizgeyi programda inşa et.
v: köşe ekle
e: kenar ekle
